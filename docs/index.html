Features

반복을 줄이고
흐름을 유지하세요

매번 맥락을 복사·붙여넣기하고 프롬프트를 다시 쓰는 번거로움 없이, 미리 설정한 Assistant가 그 역할을 대신합니다.

🤖

다중 Assistant

문서 요약, 계약 검토, 블로그 작성, 개발 리뷰 등 목적별 Assistant를 별도로 만들어 관리할 수 있습니다.

📄

페이지 컨텍스트 자동 수집

현재 페이지의 제목, URL, 선택한 텍스트, 주변 문단을 자동으로 수집해 AI에 전달합니다.

📚

참고 문서 연동

Assistant마다 참고 문서를 등록해 로컬 컨텍스트로 활용할 수 있습니다. 벡터 검색 없이 키워드 기반으로 동작합니다.

✏️

선택 영역 직접 교체

생성된 응답을 클립보드에 복사하거나, 선택한 텍스트를 AI 결과로 바로 교체할 수 있습니다.

⚡

커스텀 기능 버튼

자주 쓰는 작업(요약, 리뷰, 번역, 재작성 등)을 버튼으로 등록해두고 한 번에 실행합니다.

🔒

완전 로컬 저장

모든 설정과 API 키는 Chrome 로컬 스토리지에만 저장됩니다. 외부 서버로 전송되지 않습니다.

How it works

7단계로
완료되는 워크플로우

탭 전환 없이 사이드패널 안에서 모든 과정이 이루어집니다.

1

페이지 열기

분석하거나 작업할 웹 페이지를 Chrome에서 엽니다.

2

확장 프로그램 아이콘 클릭

툴바의 AZ-Assistant.ikkcu 아이콘을 클릭해 사이드패널을 엽니다.

3

Assistant 선택

상황에 맞는 Assistant를 선택합니다. (예: 계약 검토 / 문서 요약 / 개발 리뷰)

4

기능 선택

Assistant에 등록된 기능 버튼 중 하나를 선택합니다.

5

텍스트 선택 (선택 사항)

기능이 선택 영역을 필요로 하면 페이지에서 원하는 텍스트를 드래그합니다.

6

프롬프트 전송

자동 생성된 프롬프트를 확인하고 필요 시 수정한 뒤 전송합니다.

7

결과 활용

Markdown으로 렌더링된 응답을 복사하거나, 선택한 텍스트를 결과로 바로 교체합니다.

Setup

빠른 설치

빌드 후 Chrome에 로드하면 바로 사용할 수 있습니다.

① 저장소 클론 및 빌드

# 저장소 클론
git clone git@github.com:OVERockq/AZ-Assistant.ikkcu.git
npm install
npm run build

② Chrome에 로드

# chrome://extensions 열기
1. 개발자 모드 활성화
2. "압축해제된 확장 프로그램 로드"
3. dist/ 폴더 선택

③ Ollama 사용 시 CORS 설정

# macOS (Ollama 앱)
launchctl setenv OLLAMA_ORIGINS 'chrome-extension://*,http://localhost:*'

④ ZIP 패키징 (선택)

# dist 폴더를 ZIP으로 압축
cd dist
zip -r ../az-assistant-ikkcu.zip .
주의: API 키는 Chrome 로컬 스토리지에만 저장됩니다. 공유 빌드에 API 키를 포함시키지 마세요. Google Docs, Notion 등 복잡한 에디터에서는 텍스트 교체가 지원되지 않을 수 있으며, 이 경우 응답이 클립보드에 복사됩니다.
